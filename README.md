# AI Agent - Универсальный ИИ Агент

## Overview

Универсальный ИИ-агент с поддержкой:
- LLM (локальный или API)
- Workflow-движок
- Инструменты для работы с данными
- Визуализация результатов

## Цель

Создать переиспользуемую платформу для развёртывания ИИ-агентов на VPS с гибкой конфигурацией.

## Архитектура


## Компоненты

### 1. LLM Provider
- Поддержка локальных моделей (llama.cpp, Ollama)
- Поддержка API (OpenAI, Anthropic, и т.д.)
- Конфигурируемо через YAML/JSON

### 2. Workflow Engine
- Определение цепочек действий
- Условия и переходы
- Сохранение состояния

### 3. Tools
- **Data Tools**: чтение файлов, CSV, JSON, SQL-запросы
- **Web Tools**: HTTP-запросы, парсинг
- **Viz Tools**: генерация графиков, отчётов

### 4. Memory
- Краткосрочная (в рамках сессии)
- Долгосрочная (векторная БД)

## Deployment

- Docker для простого развёртывания
- Docker Compose для зависимостей
- Конфигурация через environment variables

## Варианты реализации

| Вариант | Сложность | Время | Описание |
|---------|-----------|-------|----------|
| **Базовый** | Простая | 1-2 дня | Python + Ollama + memory + базовые tools |
| **Средний** | Средняя | 1 неделя | + workflow-движок |
| **Продвинутый** | Сложная | 2-3 недели | Полная архитектура с визуализацией |
